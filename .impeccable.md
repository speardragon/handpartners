## Design Context

### Users

VC/AC 심사위원이 회의실에서 노트북으로 사용하는 심사 플랫폼. 심사 현장에서 PDF 서류를 보며 점수를 매기는 업무 도구. 집중력과 빠른 판단이 필요한 환경이므로 인터페이스가 방해가 되면 안 됨.

### Brand Personality

현대적 · 효율적 · 깔끔

도구적 성격이 강함. 화려함보다 명확함, 장식보다 기능. 심사위원이 "이 도구는 잘 만들어졌다"고 느끼는 수준의 품질.

### Aesthetic Direction

- **스타일**: 현재 shadcn/ui new-york 기반 유지. 미세 개선 수준.
- **테마**: 라이트 모드 전용
- **폰트**: Pretendard Variable (한국어 최적화). 단일 폰트 패밀리, weight 변화로 위계 표현.
- **컬러**: 기본 zinc 뉴트럴 팔레트. 브랜드 컬러 `ktb_navy: #0E1E46`은 로고/강조에만 사용.
- **참고하지 말 것**: 과한 그라데이션, 글래스모피즘, 네온 액센트. 이 플랫폼은 "멋져 보이는 것"이 아니라 "잘 작동하는 것"이 목표.

### Design Principles

1. **명확한 위계**: 심사위원이 0.5초 안에 어디를 봐야 하는지 알 수 있어야 함
2. **최소 인지 부하**: 불필요한 장식 없이, 콘텐츠와 액션에 집중
3. **일관된 패턴**: 같은 유형의 인터랙션은 항상 같은 방식으로 동작
4. **정보 밀도 균형**: 회의실 노트북 환경에 맞는 적절한 정보량. 너무 넓지도 좁지도 않게
5. **신뢰감**: B2B 도구답게 안정적이고 예측 가능한 인터페이스
